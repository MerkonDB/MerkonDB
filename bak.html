Features

Unparalleled Security & Performance

Built from the ground up with cryptographic verification at its core, MerkonDB provides enterprise-grade security without compromising on performance.

🔐

Cryptographic Integrity

Every database operation is cryptographically verifiable using Merkle tree proofs, ensuring absolute data integrity and authenticity across all transactions.

👥

Enterprise RBAC

Sophisticated role-based access control with fine-grained permissions, user hierarchies, and audit trails for compliance and governance.

⚡

High Performance

Optimized PMAT architecture delivers O(log k) search complexity and O(k) insert/delete operations while maintaining cryptographic proofs.

🔍

Verifiable Proofs

Generate and verify compact cryptographic proofs for any data operation, enabling trustless verification of database state and history.

💾

Persistent Storage

Reliable save/load operations with integrity checks, compaction for optimization, and support for multiple named databases.

Architecture

Designed for Security & Scale

MerkonDB's innovative architecture combines cryptographic security with high performance through its patented PMAT structure.

Core Components

Client Shell (Interactive CLI with MQL support)
Socket API (JSON over TCP)
Authentication & RBAC Module
Database Engine (Query Processing)
PMAT Indexing Structure (Partitioned Merkle Array Tree)
File Storage (Persistent Data)

PMAT Structure

The Partitioned Merkle Array Tree is the heart of MerkonDB, providing:

  • Efficient search with O(log k) complexity (k = n/partitions)
  • Fast modifications with O(k) insertion and deletion operations
  • Compact cryptographic proofs for any operation
  • Distributed-ready architecture for verifiable operations

Installation

Get Started with MerkonDB

Install and run MerkonDB in minutes with these simple steps.

Prerequisites

  • GCC compiler
  • json-c library
  • pthread library
  • Python 3 with colorama package

Build Server

gcc -o server server.c smt.c smt_db.c rbac.c -ljson-c -lpthread

Install Client

pip3 install colorama

Start Server

./server <port>

Connect Client

python3 client.py <host> <port> <username> <password>
