Features

Enterprise-Grade Database Security

MerkonDB combines cryptographic security with high performance through its innovative PMAT architecture.

🔐

Cryptographic Verification

Every database operation generates cryptographic proofs using Merkle tree architecture, ensuring verifiable data integrity and authenticity.

👥

Fine-Grained RBAC

Comprehensive role-based access control with permission bitmasks (read, write, create, delete, admin) and user hierarchies.

⚡

Optimized PMAT Structure

Partitioned Merkle Array Tree delivers O(log k) search complexity and O(k) insert/delete operations while maintaining cryptographic proofs.

🔍

Membership Proofs

Generate and verify compact cryptographic inclusion proofs for any key-value pair in the database.

💾

Multi-Database Support

Organize data across multiple named databases with collections, each maintaining independent cryptographic integrity.

📊

Comprehensive Statistics

Detailed metrics on database and collection usage with integrity verification capabilities.

Installation

Get Started with MerkonDB

Install and run MerkonDB in minutes with these simple steps. Available for Linux and macOS systems.

1

Install Prerequisites

Ensure you have the required dependencies installed:

Terminal
# For Debian/Ubuntu sudo apt-get install gcc libjson-c-dev python3 python3-pip # For macOS (Homebrew) brew install gcc json-c python
2

Build the Server

Compile the MerkonDB server with cryptographic libraries:

Terminal
git clone https://github.com/MerkonDB/MerkonDB.git cd MerkonDB gcc -o server server.c smt.c smt_db.c rbac.c -ljson-c -lpthread -O3
3

Set Up Client

Install the Python client dependencies:

Terminal
pip3 install colorama # Optional: Enhanced terminal pip3 install pyreadline
4

Launch the System

Start the server and connect with the client:

Terminal
# Terminal 1 - Server ./server (host) 8080 # Terminal 2 - Client python3 client.py (host) 8080 (user) (pass)
